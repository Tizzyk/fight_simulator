How It Works

Take the following steps to start simulating matchups

  • Select a gender. Matchups can be between any weight class but they are restricted to fighting the same gender.
  • Once the gender is chosen the fighters can be filtered by promotion and weight class, but this is only necessary if you want to further filter your results. The fighter list will populate once a gender has been selected.
  • After you have selected a fighter for each corner, click the 'Fight' button to see the results! Simple as that!
